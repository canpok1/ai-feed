手順に従って作業してください。

# 手順
1. プルリクエストのコメントを確認して作業計画を立てる
    - スラッシュコマンド `/plan-pr $ARGUMENTS` を実行すること
2. 計画を実行する
    - スラッシュコマンド `/run` を実行すること
3. PR説明文を更新する
    - document-specialistエージェントを起動して、対応内容を含むPR説明文を生成すること
    - `update-pr` スキルを使用して、生成されたPR説明文をプルリクエストに反映すること
4. プルリクエストのコメント対応が完了したことをユーザーに通知し、確認を依頼する
